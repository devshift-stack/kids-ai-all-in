# Kids AI Parent Dashboard

Parent Dashboard for monitoring children's learning progress in Kids AI apps.

## Getting Started

```bash
flutter pub get
flutter run
```

## Part of Kids-AI-Train

- [Kids-AI-Shared](https://github.com/devshift-stack/Kids-AI-Shared) - Shared components
- [Kids-AI-Train-Alanko](https://github.com/devshift-stack/Kids-AI-Train-Alanko) - Alanko App
- [Kids-AI-Train-Lianko](https://github.com/devshift-stack/Kids-AI-Train-Lianko) - Lianko App

## Features

- Monitor children's learning progress
- View activity statistics
- Manage child profiles
- Set learning goals
